| Command                     | Explanation                                      | Example                                 |
|-----------------------------|--------------------------------------------------|-----------------------------------------|
| `git add <file-name>` | Add the newly created or modified file to the staging area | `git add index.html`          |
| `git commit -m "<commit msg>"` | Commit the staged changes with a message . -m (commit msg)| `git commit -m "Added homepage"`        |
| `git status`                | Show the working tree status                     | `git status`                            |
| `git push`                  | Push changes to the remote repository            | `git push origin main`                   |
| `git pull`                  | Fetch and merge from the remote repository       | `git pull origin main`                   |
| `git clone <link>;`| Clone a repository into a new directory          | `git clone https://github.com/user/repo.git` |
| `git branch`                | List all branches                                | `git branch`                             |
| `git checkout <branch-name>;`| Switch to a branch                               | `git checkout feature-branch`            |
| `git merge <branch-name>`   | Merge a branch into the current branch           | `git merge feature-branch`               |
| `git log`                   | Show commit logs                                 | `git log`                      |
